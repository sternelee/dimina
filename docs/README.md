# Dimina 文档

本目录面向希望了解 Dimina 架构、工程构建方式，或准备参与贡献的开发者。

如果你打算为 Dimina 做贡献，建议先阅读[技术细节](./Architecture-Details.md)，了解小程序的整体原理和运行链路；随后参考[能力参考指南](./API-Reference.md)，确认当前 API、组件和平台支持状态，再补全各端缺失能力。

- [技术细节](./Architecture-Details.md)：离线包加载、逻辑视图分离、原生能力调用等实现说明。
